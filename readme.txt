*  Форматирование исходных текстов (MK61AVR Coding convention)  *
*                                                               *
*****************************************************************

1. Вместо символов табуляции используется "пробел"

2. Отступ в функциях 4 пробела:

    void About(void)
    {
        int i = 0;
    ^^^^

3. Открывающие и закрывающие скобки на отдельных строках:

    if (i = 0)
    {
        ...

    }


4. Условные операторы и т.п. всегда имеют парные скобки:

    if (i > 200)
    {
    i = 0;
    }

5. Формат условных операторов:

    if (rs)
    {

    }
    else
    {

    }

6....
